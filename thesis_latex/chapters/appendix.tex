\Cref{tab:app_xgb_cpugpu} gives exact values for the XGBoost \gls{cpu} vs.\ \gls{gpu} comparison discussed in \Cref{sec:results_xgb_cpugpu}; \Cref{fig:xgb_cpugpu} visualises the same data. Both variants implement the same algorithm and produce equivalent predictive performance --- any sub-0.001 difference in WF1 across runs is attributable to floating-point precision differences between \gls{cpu} (float64) and \gls{gpu} (float32) arithmetic and carries no practical meaning.
